# Nigerian Stock Exchange (NSE) Market Analysis 🇳🇬📈

Exploratory data analysis on the Nigerian Stock Exchange 
market index using Python and Pandas.

## Dataset
- NSE historical market index data
- Columns: Date, Price, Open, High, Low, Volume, Change %

## What I Did So Far

- Cleaned the Price column (removed commas for proper formatting)
- Converted Date column to datetime format
- Calculated average market price per year
